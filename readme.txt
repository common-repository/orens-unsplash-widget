== Description ==

Quickly display your Unsplash photos inside WordPress widget.

This plugin uses the [Unsplash API](https://unsplash.com/developers) to fetch the latest photos from your desired profile.

Read more about [Unsplash's terms](https://unsplash.com/terms), [API terms](https://unsplash.com/api-terms), and [privacy policy](https://unsplash.com/privacy).

== Features ==

* Change thumbnail size
* Caching system integrated for better performance


Unsplash Widget plugin is created by [Oren Yomtov](https://orenyomtov.com), and built upon the prior work of [Meks](https://mekshq.com)

== Live example? ==
You can see Simple Unsplash Widget live example on my [website](https://orenyomtov.com)

== Installation ==

1. Upload unsplash-widget.zip to plugins via WordPress admin panel or upload unzipped folder to your wp-content/plugins/ folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to Appearance -> Widgets to add and setup the widget
